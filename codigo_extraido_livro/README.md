# Código extraído do PDF

Esta pasta contém os blocos numerados de código do livro, separados por capítulo e listagem.

- A numeração dos arquivos segue `capítulo_listagem_assunto.py`.
- As quebras de linha, aspas tipográficas, notação científica e indentação foram normalizadas para que os arquivos sejam sintaticamente válidos em Python.
- Os exemplos não são todos programas independentes. Vários dependem de classes definidas em listagens anteriores, hardware específico, APIs externas ou execução distribuída.
- `03_extra_flash_sdp.py` é um bloco curto sem legenda própria, localizado entre as listagens 3.2 e 4.1.
- O projeto executável está em `src/`; ele reorganiza os conceitos centrais em módulos testáveis.

Consulte `INDICE_CODIGOS.md` para páginas e ordem de estudo.
